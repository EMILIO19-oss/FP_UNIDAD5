# FP_UNIDAD5
Objetivos de la App para Datos de Vuelo de Drone
1. la recoleccion de datos geoespaciales
- obtener mapas topograficos y modelos digitales del terreno
2. Informacion meteorologica en tiempo real
  -Consultar condiciones climaticas actuales (viento, lluvia, visibilidad).
  -Prever cambios meteorológicos que puedan afectar el vuelo.
3. Validación de Zonas de Vuelo
  -Consultar regulaciones locales de espacio aéreo y permisos necesarios.
